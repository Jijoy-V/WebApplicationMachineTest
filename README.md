
Development Steps - Machine Test MVC
--------------------------------------

1. Functional Requirements :

	Read and analyze all the requirements.

2. Tables and Stored Procedures :

	Create a Relational Database with tables for login authentication and data storage.	
	Create Stored Procedures for making CRUD Operations.

	Role Table , User Table , Student Table, Students Record Table

3. Add NuGet Packages and dll file :
	Install the wanted packages to include in the project and add database connection dll file as an COM Reference

4. Models and ViewModels:
   Create Models and ViewModels for the data view

5. Repository :
	Create a repository layer for Connection with database and to get stored procedures
	and to include all the methods

6. Service : 
	Create a service layer to implement all the business logic.

7. Controller :
	Create a controller to include all the action methods .

8.  Index.cshtml :  
	Create view for controller index.cshtml with modal forms for add , update and delete
